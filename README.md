# Simple To-Do App (DOM Manipulation Project)

A lightweight To-Do application built using plain JavaScript, HTML, and CSS, demonstrating DOM manipulation, event handling, and basic state management.

## Live Demo

Check out the live version here: https://movnisha.github.io/Simple-To-Do-App-DOM-Manipulation-Project/

## Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Filter tasks by:
  - All
  - Completed
  - Pending
- Responsive design for both desktop and mobile views

> **Note**: This version does not include LocalStorage, so tasks will be cleared when the page is refreshed.

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- DOM Manipulation

## Getting Started

To get a copy of the project running locally, follow these steps:

1. Clone the repository:

    ```bash
    git clone https://github.com/Movnisha/Simple-To-Do-App-DOM-Manipulation-Project.git
    ```

2. Open `index.html` in your browser, or use the live demo link to try it out.

3. Start adding, deleting, and filtering tasks!
